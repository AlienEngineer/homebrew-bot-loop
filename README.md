# homebrew-bot-loop

Homebrew tap for [bot-loop](https://github.com/AlienEngineer/bot-loop).

```sh
brew tap alienengineer/bot-loop
brew install bot-loop
```

`Formula/bot-loop.rb` is **auto-maintained**: the `release` workflow in the
main repo rebuilds it on every release and mirrors it here. Do not edit it by
hand.

> Note: installs succeed only once a `vX.Y.Z` release with a `bot-loop-macos.tar.gz`
> asset exists and the formula's `url`/`sha256` have been repointed at it by CI.
